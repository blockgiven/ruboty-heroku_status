# Ruboty::HerokuStatus

Ruboty plugin for ちょっとHerokuの様子見てくる。

![screenshot](screenshot.png)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-heroku_status'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-heroku_status
